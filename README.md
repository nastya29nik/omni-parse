# OmniParse

**OmniParse** — это stateless-микросервис для потоковой конвертации документов, парсинга сложных форматов и распознавания текста в реальном времени.

---

## Архитектурные особенности
* Полный отказ от сохранения временных файлов на жесткий диск. Вся обработка и трансформация данных происходит исключительно в оперативной памяти через `InputStream` и `ByteArrayOutputStream`.
* Разделение ответственности по доменным сервисам и централизованная обработка исключений.
* Безопасное управление жизненным циклом потоков ввода-вывода с помощью `try-with-resources`.

## Стек технологий
- **Язык:** Java 17
- **Фреймворк:** Spring Boot 4
- **Обработка документов и PDF:** Apache PDFBox, OpenHTMLToPDF, Apache POI
- **Распознавание текста:** Tesseract OCR (Tess4J)
- **Документация API:** OpenAPI 3, Swagger
- **Сборка и управление проектом:** Apache Maven
- **Контроль версий:** Git

## Доступные REST API эндпоинты

| Метод | Эндпоинт | Описание                                                           |
| :--- | :--- |:-------------------------------------------------------------------|
| `POST` | `/api/convert/html-to-pdf` | Конвертация HTML-верстки в чистый векторный PDF                    |
| `POST` | `/api/convert/docx-to-pdf` | Конвертация документа Word (`.docx`) в PDF                         |
| `POST` | `/api/convert/docx-to-txt` | Извлечение текста из документа Word                                |
| `POST` | `/api/convert/image-to-pdf` | Упаковка графического файла (JPEG, PNG, BMP) в PDF по его размерам |
| `POST` | `/api/convert/image-to-txt` | Распознавание текста со скриншотов и фотографий                    |
